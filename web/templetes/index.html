Project KAAU.

Kubernetes Authentication Authorisation Utility

This is a Hobby Project yet not completed. This project aims to provide a management utility to Authorisation and Authentication for Kubernetes from a simple UI

Read bit about RBAC On Medium

Status of Project. Currently, it can Authentication via web. you can see password hard coded in project ;-) It can connect to Kubernetes via kubernetes client.go library

It can do all (Create, Read, Update, and Delete) on Kubernetes Authentication and Authorization part which are.

  • service Account
  • Roles
  • Cluster Roles
  • RoleBinding
  • ClusterRoleBinding
